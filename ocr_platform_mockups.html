OCRouter
◆ The OCR intelligence layer

Every OCR engine. One intelligent answer.

Upload a document and our agent benchmarks every engine — open-source and commercial — on your actual data, then routes to the winner. Stop guessing which OCR to use.

quickstart.py 
# One call. We pick the best engine for each document.
from ocrouter import Client

client = Client(api_key="sk_live_...")

result = client.recognize(
    file="invoice_urdu.pdf",
    strategy="auto",   # agent chooses + routes
    priorities={"accuracy": 0.7, "cost": 0.3}
)

print(result.engine)      # → "paddleocr-vl"
print(result.text)        # extracted text
print(result.confidence)  # → 0.96
